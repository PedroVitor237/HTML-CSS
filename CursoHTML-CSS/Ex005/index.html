Itálico / Ênfase

Nesta frase temos um termo em itálico utilizando a tag i (não semântica).

Nesta frase temos um termo em itálico utilizando a tag em (semântica).

Texto Marcado

Nesta frase temos um termo marcado como um "marca texto" utilizando a tag mark.

Texto deletado

Nesta frase temos um texto deletado utilizando a tag del para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado posteriormente.

Existe também o sublinhado com a tag u (não semântica)

Texto sobrescrito

Podemos usar a tag sup para sobrescrever um texto, como em expoentes matemáticos: x2+3

Texto subscrito

Podemos usar a tag sub para subscrever um texto, como em fórmulas químicas: H2O

Código fonte & Pré-formatação

Para escrever um código fonte, usamos a tag code como em console.log("Hello World")

Para pré-formatar um texto, usamos a tag pre como em: 

        
            num = int(input("Digite um número: "))
            if num % 2 == 0:
                print(f'O número {num} é par')
            else:
                print(f'O número {num} é ímpar')

Citações simples

Podemos usar a tag q para citações simples, como: Essa é uma citação simples.
