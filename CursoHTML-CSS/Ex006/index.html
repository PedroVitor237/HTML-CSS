Listas

Listas Ordenadas

As listas ordenadas são usadas quando a ordem dos itens é importante. Elas são numeradas automaticamente pelo navegador.

  1. Primeiro item
  2. Segundo item
  3. Terceiro item
  4. Quarto item
  5. Quinto item

Listas Não Ordenadas

As listas não ordenadas são usadas quando a ordem dos itens não é importante. Elas utilizam marcadores (bullets) para indicar cada item.

  • Primeiro item
  • Segundo item
  • Terceiro item
  • Quarto item
  • Quinto item

Listas de Aninhadas

Listas aninhadas são listas dentro de outras listas, ordenadas ou não. Elas podem ser usadas para criar hierarquias ou subcategorias.

  1. Item 1
    • Subitem 1.1
    • Subitem 1.2
  2. Item 2
    • Subitem 2.1
    • Subitem 2.2
  3. Item 3

Lista de Definições

Listas de definições são usadas para definir termos ou conceitos. Elas consistem em pares de termos e suas definições.

HTML
HyperText Markup Language: Linguagem de Marcação de Hipertexto usada para estruturar páginas web.
CSS
Cascading Style Sheets: Folhas de Estilo em Cascata usadas para estilizar páginas web.
JavaScript
Linguagem de programação usada para criar interatividade em páginas web.
HTTP
HyperText Transfer Protocol: Protocolo de Transferência de Hipertexto usado para comunicação entre clientes e servidores web.
