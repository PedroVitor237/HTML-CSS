Parágrafos e quebras de linha

Este é o primeiro parágrafo da página. Parágrafos são usados para organizar o texto em blocos distintos, facilitando a leitura e a compreensão do conteúdo.

Este é o segundo parágrafo. Note que há um espaço entre os parágrafos, o que ajuda a separar visualmente as ideias apresentadas.

É possível escrever um parágrafo de qualquer jeito, basta estar entre as tags <p> e </p>.

Se quisermos criar uma quebra de linha dentro de um parágrafo, podemos usar a tag <br> para indicar onde a linha deve ser quebrada.
